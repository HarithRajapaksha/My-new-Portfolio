Featured Projects

The Karnivorse Resturant Payroll Manegement system

This system manages employee attendance using barcodes. When an employee registers, they are assigned a unique ID linked to their barcode. Attendance, half-day, and no-pay details are automatically tracked, and EPF/ETF calculations are done automatically. Employees can apply for half-day and full-day leave through the system. It uses role-based access control, with all requests handled through the admin panel. The admin dashboard displays attendance, leave, and other data in graphs, and reports can be downloaded as PDFs.

Automated Wayamba university Medical center

Developed an automated medical center system for Wayamba University, replacing the traditional manual process. The system efficiently manages student registration details, allowing them to be retrieved quickly and shared with doctors for consultations. It records and organizes medical information, tracks patient history, and forwards prescriptions directly to the pharmacist. This platform streamlines the entire medical workflow, improves operational efficiency, reduces errors, and enhances overall healthcare service delivery for students and staff.

Gem Identification System_CNN Model

Developed a Gem Identification System using Convolutional Neural Networks (CNN) with TensorFlow to automatically recognize and classify gemstones. The system takes an input image of a gem, processes it through the trained neural network, and accurately identifies its type, name, and characteristics. This project demonstrates the practical application of deep learning in image recognition, providing a fast, reliable, and user-friendly tool for gem classification and educational purposes.

Feld Mobile app

This project is designed to support people with disabilities by ensuring their safety and health through continuous daily activity monitoring.
